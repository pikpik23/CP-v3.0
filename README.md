## Development of CP 3.0
**Prerequisites**
* [Git](https://git-scm.com/)
* [Pip](https://pip.pypa.io/en/stable/installing/)

**Installation**
* `git clone https://github.com/pikpik23/CP-v3.0/`
* `cd CP-v3.0/CP\ Software`
* `pip3 install -r requirements.txt`

**Running**
* `./app.py`
